# brAInstorm

> *...or how I learned to stop worrying and love building again*

**brAInstorm** is a project idea generator app, that gives you interesting, fun and challenging project ideas to work on. It is the result of many sleepless nights thinking about building, but not actually getting around to it, and many other nights looking for inspiration.

Ironically enough it's what came to my mind when I started prompting chatbots for project ideas, with detailed instructions (i.e. give me actually interesting project ideas, write an MVP, don't hallucinate, etc.) and it's perfect - solves an actual problem I've encountered, and lets me learn more about full-stack web development, AI, Python, deployment etc.
